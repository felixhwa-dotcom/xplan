四、Q&A 常见问题汇总（完整版 · 共30问）

🌟 星火计划 Q&A 15问

Q1：任务ID和密钥从哪获取？
淘宝星河平台入驻并开通种草自选权限 → 服务商身份接单 → 订单管理-订单ID处获取任务ID → 订单详情获取密钥。
⚠️ 服务商主体需与花火发布商单主体一致。
Q2：花火平台如何操作关联星火任务？
申请独立星火账户（命名：xx公司-商单-花火平台-淘联），在【数据合作】模块选择合作平台为"星火"，填写任务ID和密钥。无此板块可联系渠道经理申请开白。
Q3：星火计划的发票如何开具？
同普通商单。【我的花火】→【财务管理】→【申请开票】，结算单类型选【星任务结算单】，核对明细后打印盖章上传，填写发票抬头后提交审核。
Q4：星河平台订单与花火平台任务的对应关系？
星河平台的订单列表对应花火平台的任务（通过任务ID关联），星河的达人列表对应花火该任务下的订单。
Q5：种草自选创建是否有费用门槛？
有。种草自选订单费用需 ≥ 2万元，否则无法发布。
Q6：星河平台入驻时三种身份如何选择？
品牌：广告主，拥有充值、发单、授权权限
代理商：获得品牌授权后代发单
服务商：拥有接单权限

商家花火自投需满足「品牌+服务商」；代理商代投需满足「代理商+服务商」
Q7：前后链路数据分别是什么？
前链路（花火）：播放/互动/CPM/CPE 等
后链路（淘宝/天猫）：搜索/进店/收藏/加购/关注/新访客/新客/成交等
Q8：海外店铺品牌可以通过星火计划推广吗？
可以注册品牌，但需通过代理商发单。海外商家只能注册为【品牌】身份，不能注册为【代理商】或【服务商】，投放操作仍需代理商代为操作。
Q9：星河接任务主体和花火下单方可以是不同主体吗？
不可以。种草自选接任务主体必须跟花火发布任务主体保持一致，否则无法下单。
Q10：稿件上线后何时可以看到后链路数据？
T+1 可在星河查看任务组维度数据；T+1 23:00 可在花火投后报告-效果转化查看单稿件数据。
Q11：星火计划的账户如何进行充值？
花火平台无需充值，下单时关联种草自选订单ID，占用其预算。需在星河平台发布任务前先充值：
路径：【财务管理】→【账户明细】→【立即充值】→ 填写金额确认（仅支持支付宝）
Q12：「花火+」权益如何收费？
星火计划订单目前「花火+」免费使用，无需额外付费。
Q13：评论区商品链接的蓝色标题如何修改？
① 悬赏带货配置的 → 悬赏后台修改
② 三连后台创建的 → 三连后台-商品管理中自助修改
Q14：评论区如何配置挂载多个商品链接？
① UP主自助手动添加
② 代理商/品牌主在订单详情页转化组件板块中添加
Q15：稿件发布后能否补绑任务ID？
❌ 不支持补绑定！必须在电商平台发布种草任务拿到ID和密钥后，在花火发布商单任务时关联。未关联则无法数据回传！请务必先拿ID再发布稿件。

🔥 京火计划 Q&A 15问

Q1：任务ID和密钥从哪获取？
完成京准通+京东联盟入驻 → 京准通新建种草任务并签署协议 → 品牌或代理商完成接单 → 任务详情获取ID和密钥。⚠️ 接任务主体需与花火发布主体一致。
Q2：花火平台如何操作关联京火任务？
申请独立京火账户（命名：xx公司-商单-花火平台-京火），在【数据合作】模块选择"京东联盟"，填入任务ID和密钥。无此板块可联系渠道经理申请开白。
Q3：京火计划的发票如何开具？
同普通商单。【我的花火】→【财务管理】→【申请开票】，结算单类型选【花火结算单】，核对明细后打印盖章上传，填写发票抬头后提交审核。
Q4：京东POP商家是否支持京火计划？
支持。可通过商家发布任务、服务商接收任务的模式进行合作。商家需先在京准通完成入驻。
Q5：京火任务是否有费用门槛？
有。单任务费用 ≥ 2万元，否则无法发布。
Q6：服务商报名了但后台没有种草任务是为什么？
当品牌发布的是公开任务时，服务商报名后，品牌方需点击【接受】才能达成合作，之后才能看到种草任务并获取任务ID和密钥。
Q7：自发自接 vs 自发他接 有什么区别？
自发自接：自己发自己接 → 无需接单步骤 → 自动生成任务组ID及密钥
自发他接：品牌发、代理商接 → 代理商报名 → 品牌确认后才达成合作
Q8：一个任务的有效期是多少天？到期可以延期吗？
任务有效期最长 90天，到期后不可以延期。
Q9：任务时间是否可以修改？
商家发-服务商接模式：被接单前可更改，接单后不可改
自发自接模式：已发完的任务不可更改，如需更改可重新发任务。
Q10：首次发布任务时没签署协议，在哪里签署？
商家：京准通 → 内容广告 → 商家任务 → 站外任务 → 新建种草任务 → 弹出协议点击同意
服务商：登录京东联盟 → 我的推广 → 京红任务 → 协议签署
Q11：京火计划的账户如何进行充值？
京火账户充值流程同普通账户一致。参考：代理商充值 / 品牌主充值
Q12：「花火+」权益如何收费？
若品牌已享受花火政策 → 按2%收费
若品牌未申请花火政策 → 按5%收费
具体政策申请请联系对接销管沟通。
Q13：评论区商品链接的蓝色标题如何修改？
① 悬赏带货配置的 → 悬赏后台修改商品名
② 三连后台配置的 → 三连后台-商品管理中自助修改
Q14：评论区如何配置挂载多个商品链接？
① UP主自助手动添加
② 代理商/品牌主在订单详情页转化组件板块中添加
Q15：稿件发布后能否补绑任务ID？
❌ 不支持补绑定！必须在电商平台发布种草任务拿到ID和密钥后，在花火发布商单任务时关联。未关联则无法数据回传！请务必先拿ID再发布稿件。
